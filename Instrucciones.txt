Instrucciones para Ejecutar:


1- Clonar el repositorio con el comando:

git clone https://github.com/farinavic/Spoty-ETL.git
cd Spoty-ETL

2- Ingresar credenciales en los archivos de la carpeta del contenedor de Spoty-ETL/keys corroborando que no tengan comillas, espacios, saltos de linea o cualquier caracter extra.
Client ID de la API de Spotify: CLIENT_ID_SP.txt
Client secret  de la API de Spotify: CLIENT_SECRET_SP.txt
Usuario de base de datos Redshift: RS_USER.txt
Contraseña de base de datos Redshift: RS_PWD.txt


3- Construir e iniciar los servicios con Docker Compose usando el siguiente comando:

docker-compose up --build



4- Acceder a la Interfaz de Airflow

Abrir el navegador e ir a http://localhost:8080. Iniciar sesión con las credenciales airflow/airflow.
